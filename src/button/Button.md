Button example:

```js
import { Button, Row, Col } from 'zk-ui';

<Row>
    <Col>
        <Button>
            Button
        </Button>
    </Col>
    <Col>
        <Button disabled>
            Button disabled
        </Button>
    </Col>
    <Col>
        <Button primary theme='green'>
            Primary green
        </Button>
    </Col>
    <Col>
        <Button primary theme='orange'>
            Primary orange
        </Button>
    </Col>
    <Col>
        <Button primary theme='green' disabled>
            Primary green disabled
        </Button>
    </Col>
</Row>
```
